# NGO MERN Web App

A modern MERN starter for an NGO website with:

- React + Vite frontend
- Express + MongoDB backend
- NGO-focused landing page sections
- Sample impact, programs, and volunteer endpoints

## Run locally

1. Install dependencies:
   - `npm install`
   - `npm install --workspace client`
   - `npm install --workspace server`
2. Create `server/.env` from `server/.env.example`
3. Start the app:
   - `npm run dev`

## Project structure

- `client` for the React frontend
- `server` for the Express API
